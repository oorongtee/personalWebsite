# Ray's Personal Website

一個現代化的個人作品集網站，使用 Vanilla JavaScript 打造，具有豐富的視覺效果和互動功能。

## ✨ 特色功能

- **響應式設計**: 完美適配各種螢幕尺寸
- **單頁應用**: 流暢的路由切換體驗
- **AI 聊天助理**: 智能個人助理功能
- **動態天氣效果**: 整合真實天氣數據的視覺效果
- **美麗的動畫**: 包含樹葉動畫、風紋效果和背景色塊
- **大地色調**: 溫暖舒適的配色方案

## 🛠️ 技術棧
- **HTML5** - 語義化標記
- **CSS3** - 自定義樣式，包含動畫和響應式設計
- **JavaScript (ES6+)** - 動態路由和互動功能
- **台灣中央氣象署 API** - 真實天氣數據整合
- **EmailJS** - 聯絡表單功能

## 主要功能

### 📱 響應式設計
- 支援桌面 (1024px+)
- 平板 (768px-1024px) 
- 手機 (360px-768px)
- 小螢幕 (360px 以下)

### 🎯 頁面功能
- **Home** - 個人介紹和作品展示卡片
- **About** - 專業經歷和技能展示
- **Works** - 專案作品集，含互動式詳情面板
- **Notes** - 文章列表，支援搜尋和分類篩選
- **Contact** - 聯絡表單，整合 EmailJS

### ✨ 互動特效
- 動態導航指示器
- 霧面玻璃效果 (backdrop-filter)
- 滑鼠懸停動畫
- 平滑滾動和過渡效果
- **背景動畫**: 多層次的色彩漸層和動態效果
- **天氣整合**: 根據真實天氣數據顯示不同的視覺效果
- **互動樹葉**: 可點擊的樹葉顯示天氣資訊
- **風力效果**: 動態風紋根據風速變化

### 🔧 技術特色
- **Hash 路由系統** - 單頁應用架構
- **動態內容渲染** - JavaScript 模板系統
- **模組化 CSS** - 組件化樣式結構
- **無框架依賴** - 純原生技術實現
- **AI 聊天系統** - 內建智能助理
- **實時天氣API** - 動態視覺效果

### 🤖 AI 助理功能
內建智能聊天助理，可以：
- 回答關於 Ray 的問題
- 提供專案和技能資訊
- 協助訪客了解更多內容

## 🚀 快速開始

1. **複製專案**
   ```bash
   git clone https://github.com/oorongtee/personalWebsite.git
   cd personalWebsite
   ```

2. **本地開發**
   ```bash
   # 使用 Python 啟動本地服務器
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```

3. **在瀏覽器中開啟**
   ```
   http://localhost:8000
   ```
